# Vircadia World Python SDK

## Install

Create and activate the development env:

```sh
python3 -m venv .venv
source .venv/bin/activate
```

Install the dependencies:

```sh
pip install -r requirements.txt
cd shared/modules/vircadia-world-meta/python
pip install -r requirements.txt
```
